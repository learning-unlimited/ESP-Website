Module Management for {{ program.niceName }}

Welcome to the module management page for {{ program.niceName }}. This page lists the modules that students and teachers will encounter during registration. You can edit individual modules by clicking on them in the timeline to set their Active Window dates, Required status, and Registration Step Title. The Active Window determines when a module is available for users to complete. Modules that are "Always On" are available for the duration of the program.

{% inline_program_qsd_block program "manage:modules" %} {% end_inline_program_qsd_block %}
Please be aware that some modules require other modules to have been completed already (e.g., teacher class registration usually requires a teacher to have already set their availability). Please use common sense and make sure to test everything whenever you change any of these settings! Feel free to contact websupport with any questions. Furthermore, the following settings will always be enforced regardless of how you edit the modules below (if they are enabled for your program):
  • Profile Editor Modules must always be first and required
  • The Availability Module must always be required
  • Acknowledgment Modules must always be required
  • The Two Phase Lottery Module must always be required
  • The Credit Card Module must always come after everything except the Confirm Registration Module and cannot be required
  • The Confirm Registration Module cannot be required
