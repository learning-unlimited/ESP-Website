Print student schedules

Using this tool you can print student schedules for students on the fly.
To use this tool as intended you must:

  1. Get a printer that can reliably sit and be plugged into a printer
  2. Install a printer, and make SURE it's default (at least in Firefox)
  3. Use firefox :-D
  4. Set up the printer as default:
    1. Goto about:config and type `print_printer'. Enter the string title (exactly) for the {{ settings.ORGANIZATION_SHORT_NAME }} Printer.
    2. Rightclick on some whitespace and click on "New --> Boolean". Add print.always_print_silent and make it TRUE.
    3. While you're at it, you might as well make print.print_color false.
    Now your printer will now automatically print from firefox. Test this by printing this page. Notice it shouldn't open a dialog box or anything.
  5. Now you should be ready to go to the next page!
MAKE SURE YOU DO THIS ON ONE COMPUTER. IF YOU DON'T KNOW WHAT YOU ARE DOING, PLEASE GO BACK NOW!
