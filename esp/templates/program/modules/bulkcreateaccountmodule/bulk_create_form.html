Bulk Account Creation Form

This form lets you bulk-create sets of (usually student) accounts with a common prefix in the username and common password. A typical use case is when admins visit a school onsite to help a large number of students register for a program at the same time. It saves a lot of time if accounts have already been created for students. Each prefix corresponds to a school.

It is recommended that you choose prefixes with the school name and the program date. For example, for school XYZ for Fall 2017, the prefix could be "xyzfa17". This is to avoid conflicts in usernames from one program to the next.

Example: a prefix of "xyzfa17" with 5 accounts would create usernames xyzfa171, xyzfa172, xyzfa173, xyzfa174, xyzfa175.

You must select either "Student" or "Teacher", in addition to any other groups you want.
